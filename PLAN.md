# 日本語コーヒー補正アプリ『次の一杯』を3画面で作成

**Features**
- 味の印象を選ぶだけで、次の一杯で変えることを1つだけ大きく提案します。
- 「よく分からない」を選ぶと、補助質問がその場で開いて判断しやすくします。
- 器具・焙煎度・お湯の落ちる速さを選ぶと、条件に合った提案を返します。
- 対応していない組み合わせでも、失敗しないように汎用的な提案を返します。
- 提案内容と選んだ条件を端末内に保存できるようにします。
- 画面保存ができない環境では、共有できる形で案内します。
- 次の一杯の結果入力は2回以内のタップで完了できる流れにします。
- 日本語を前提に、朝のキッチンで迷わず使える大きな操作にします。

**Design**
- 全体はクリーム色をベースにした、モダンでミニマルな喫茶店風にします。
- ボタンは指で押しやすい大きさにし、選んだ状態がひと目で分かる見た目にします。
- 余白を広めに取り、落ち着いたブラウン系の差し色でコーヒーらしい温かさを出します。
- 提案画面は一番大事な答えが最初に目に入るよう、強弱のある文字組みにします。
- 3画面ともスクロールなしで収まる、すっきりしたレイアウトにします。
- 小さな反応や押した感覚を入れて、単純でも気持ちよく使えるようにします。

**Pages / Screens**
- **診断画面**: 味・器具・焙煎度・お湯の落ち方を選び、次の一杯の修正に進みます。
- **提案画面**: 提案を1つだけ大きく見せ、理由・保存・画面保存または共有・次の入力への導線をまとめます。
- **結果入力画面**: 改善したかどうかをすぐ記録し、保存または再診断へ短く誘導します。

**Saved Data**
- 保存する内容は、提案文に加えて味・器具・焙煎度・速度の条件を含めます。
- ログインなしで使え、保存した内容はこの端末の中だけに残ります。

**App Icon**
- クリーム色の背景に、湯気の立つコーヒーカップと一滴のドリップを組み合わせた、静かで上質な印象のアイコンにします。
